ANIMALIA API

Module functionalities

MODULES '/modules'
lists all the available modules
method: GET      url parameters: none      return type: application/JSON

MODULE '/modules/:module'
displays info for the selected module, including a spotlight animal and a list of all the animals
method: GET      url parameters: MODULE      return type: application/JSON
example: http://hcibiology.herokuapp.com/modules/Mammals

ANIMALS '/modules/:module/animals'
displays a list of the animals in the given module
method: GET      url parameters: MODULE      return type: application/JSON
example: http://hcibiology.herokuapp.com/modules/Mammals/animals

Animal functionalities

ANIMALS '/animals'
an index of all the animals
method: GET      url parameters: none      return type: application/JSON

ANIMAL OF THE DAY '/animals/aotd'
displays info for the animal of the day
method: GET      url parameters: none      return type: application/JSON

ANIMAL '/animals/animal/:id'
displays detailed info for an animal
method: GET      url parameters: ID      return type: application/JSON
example: http://hcibiology.herokuapp.com/animals/5.1

Quiz functionalities

QUIZZES '/quizzes'
lists all the available quizzes
method: GET      url parameters: none      return type: application/JSON

QUIZ '/quizzes/quiz/:quiz'
generates a quiz with 5 random questions from the given module
method: GET      url parameters: QUIZ      return type: application/JSON
example: http://hcibiology.herokuapp.com/quizzes/quiz/Mammals

SUBMIT '/quizzes/submit'
submits the results for a given quiz
method: GET      url parameters: USERNAME, TYPE, GUESSES, TIME LEFT      return type: application/JSON
example: http://hcibiology.herokuapp.com/quizzes/submit?username=usr85f77d9a06d220334e9e9f72b3250403&type=Expert&guesses=3&timeleft=25

HIGH SCORES '/quizzes/highscores/:list'
displays the high score lists
method: GET      url parameters: LIST      return type: application/JSON
example: http://hcibiology.herokuapp.com/quizzes/highscores/Total
